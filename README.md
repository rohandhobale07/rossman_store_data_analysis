                                   📈📊 Rossmann Store Sales Analysis Dashboard 📊📈

This project focuses on performing Exploratory Data Analysis (EDA) on Rossmann HealthChain sales data using Tableau. The dashboard provides insights into sales trends, top-performing stores, customer behavior, and promotional impact through interactive visualizations.

🛠️ Tools Used:
- Tableau
- Excel / CSV Dataset

📌 Key Features:
- Quarterly sales trend analysis
- Top 10 stores visualization
- Historical sales pattern analysis
- Promotion vs non-promotion customer analysis
- Interactive Tableau dashboard

📊 Key Insights:
- Sales show clear quarterly trends.
- Promotions increase customer visits and sales.
- Some stores consistently outperform others.
- Identified least-performing stores for improvement.
- Customer behavior changes during promotional periods.
